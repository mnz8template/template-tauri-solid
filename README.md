# Tauri Solid

[Tauri](https://tauri.app/)
[Solid](https://www.solidjs.com/)

## 启动

    pnpm dev 前端启动
    pnpm tauri dev 启动 tauri 开发模式

## track

pnpm create vite

Select Solid TypeScript

修改目录 src-solid

`pnpm add -D @tauri-apps/cli`

`pnpm tauri init`

